Collab VM Default Page

Congratulations! If you're reading this, that means you did not mess up and you set Collab VM up correctly. This is the default page you see when you first set up Collab VM. You can access the administrator panel by going to /admin/. You can find the default password in the README.

Please note that you may need to change the IP Address in admin.min.js. Open a text editor, search "127.0.0.1:6004" and replace the IP with your current one. Alternatively, you can compile collab-vm-admin-webapp to use it or if 127.0.0.1 works for you, you can just leave it as default. As for this page, you can either keep it, modify it, or remove it, its your choice.
